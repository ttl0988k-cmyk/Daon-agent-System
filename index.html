🌸 DAON Dynamic Harness 사용 설명서

🤖 다이나믹 하네스(Dynamic Harness)란?

사용자의 요구사항을 해결하기 위해 백그라운드에서 실시간으로 멀티 에이전트(Multi-Agent)들이 협업하여 코딩, 실행, 오류 검증 및 자가 수정을 수행하는 최첨단 오케스트레이션 엔진입니다.

✨ 핵심 기능

  • 코드 자동 생성 및 검증: 코드를 자동으로 작성하고 실행하여 컴파일 오류나 실행 에러를 자가 감지 및 수정합니다.
  • 실시간 에이전트 로그 스트리밍: 에이전트들의 세부 작업 로그가 하단의 CLI Console에 실시간으로 출력됩니다.
  • 작업공간 자동 갱신: 에이전트의 작업이 완료되면 탐색기와 에디터에 결과가 즉시 동기화됩니다.

🚀 사용 방법

  • 방법 1 (우측 채팅 패널): 일반 대화 창에 작업 지시사항을 입력하여 자연스럽게 에이전트를 구동합니다.
  • 방법 2 (하단 CLI 콘솔): CLI 입력창에 run [작업내용]을 직접 실행합니다.
    (예시: run index.html 파일 검토해줘)
