# 인더스트리얼 브루탈리즘 & 택티컬 텔레메트리 UI

## 1. 스킬 메타

**이름:** 인더스트리얼 브루탈리즘 & 택티컬 텔레메트리 인터페이스 엔지니어링
**설명:** 중세기 스위스 타이포그래피 디자인, 산업 제조 매뉴얼, 레트로-퓨처리스틱 항공우주/군사 터미널 인터페이스를 종합하는 웹 인터페이스 설계의 고급 숙련도. 이 분야는 경직된 모듈형 그리드, 극단적인 타이포그래피 스케일 대비, 순수 실용적 색상 팔레트, 아날로그 열화(하프톤, CRT 스캔라인, 비트맵 디더링)의 프로그래매틱 시뮬레이션에 대한 절대적 숙달을 요구한다. 목표는 원시적 기능성, 기계적 정밀성, 높은 데이터 밀도를 발산하는 디지털 환경을 구축하는 것이며, 관습적인 컨슈머 UI 패턴을 의도적으로 폐기한다.

## 2. 시각적 아키타입

디자인 시스템은 두 개의 뚜렷하지만 높은 호환성을 가진 시각적 패러다임을 병합하여 작동한다. **프로젝트당 하나를 선택하고 그것에 전념하라. 동일한 인터페이스 내에서 두 모드를 교대하거나 혼합하지 마라.**

### 2.1 스위스 인더스트리얼 프린트
1960년대 기업 아이덴티티 시스템과 중장비 청사진에서 파생.
- **특성:** 고대비 라이트 모드 (신문지/오프화이트 기질). 모놀리식하고 무거운 산세리프 타이포그래피에 의존. 가시적 구분선으로 윤곽이 그려진 용서 없는 구조적 그리드. 오버사이즈되고 뷰포트를 넘나드는 숫자나 글자로 점철된 공격적이고 비대칭적인 네거티브 스페이스 사용. 경고/액센트 색상으로 프라이머리 레드의 과도한 사용.

### 2.2 택티컬 텔레메트리 & CRT 터미널
기밀 군사 데이터베이스, 레거시 메인프레임, 항공우주 HUD(Head-Up Display)에서 파생.
- **특성:** 다크 모드 전용. 고밀도 표 형식 데이터 프레젠테이션. 모노스페이스 타이포그래피의 절대적 지배. 기술적 프레이밍 장치(ASCII 브래킷, 크로스헤어)의 통합. 시뮬레이션된 하드웨어 제한(인광 글로우, 스캔라인, 낮은 비트 심도 렌더링)의 적용.

## 3. 타이포그래피 아키텍처

타이포그래피는 주요 구조적 및 장식적 인프라다. 이미지는 부차적이다. 시스템은 스케일, 웨이트, 간격의 극단적인 변화를 요구한다.

### 3.1 매크로 타이포그래피 (구조적 헤더)
- **분류:** Neo-Grotesque / Heavy Sans-Serif.
- **최적 웹 폰트:** Neue Haas Grotesk (Black), Inter (Extra Bold/Black), Archivo Black, Roboto Flex (Heavy), Monument Extended.
- **구현 매개변수:**
  - **스케일:** 유체 타이포그래피를 사용한 대규모 배치 (예: `clamp(4rem, 10vw, 15rem)`).
  - **트래킹 (Letter-spacing):** 극도로 타이트, 종종 네거티브 (`-0.03em` ~ `-0.06em`), 글리프가 단단한 건축적 블록을 형성하도록 강제.
  - **리딩 (Line-height):** 과도하게 압축 (`0.85` ~ `0.95`).
  - **케이싱:** 구조적 임팩트를 위해 독점적으로 대문자.

### 3.2 마이크로 타이포그래피 (데이터 & 텔레메트리)
- **분류:** Monospace / Technical Sans.
- **최적 웹 폰트:** JetBrains Mono, IBM Plex Mono, Space Mono, VT323, Courier Prime.
- **구현 매개변수:**
  - **스케일:** 고정적이고 작게 (`10px` ~ `14px` / `0.7rem` ~ `0.875rem`).
  - **트래킹:** 넉넉하게 (`0.05em` ~ `0.1em`) — 기계적 타자기 간격이나 터미널 매트릭스 시뮬레이션.
  - **리딩:** 표준에서 타이트 (`1.2` ~ `1.4`).
  - **케이싱:** 독점적으로 대문자. 모든 메타데이터, 내비게이션, 유닛 ID, 좌표에 사용.

### 3.3 텍스처 대비 (예술적 파괴)
- **분류:** High-Contrast Serif.
- **최적 웹 폰트:** Playfair Display, EB Garamond, Times New Roman.
- **구현 매개변수:** 극도로 드물게 사용. 과도한 후처리 적용 필수 (하프톤 필터, 1-bit 디더링) — 벡터 완벽성을 열화시켜 깨끗한 산세리프와의 텍스처 병치 생성.

## 4. 색상 시스템

색상 아키텍처는 타협하지 않는다. 그라데이션, 소프트 드롭 그림자, 현대적 반투명은 엄격히 금지된다. 색상은 물리적 매체나 원시적 발광 디스플레이를 시뮬레이션한다.

**중요: 프로젝트당 하나의 기질 팔레트를 선택하고 일관되게 사용하라. 동일한 인터페이스 내에서 라이트와 다크 기질을 혼합하지 마라.**

### 스위스 인더스트리얼 프린트 (라이트) 선택 시:
- **배경:** `#F4F4F0` 또는 `#EAE8E3` (무광, 미표백 문서 용지).
- **전경:** `#050505` ~ `#111111` (카본 잉크).
- **액센트:** `#E61919` 또는 `#FF2A2A` (항공/위험 레드). 이것이 **유일한** 액센트 색상. 취소선, 두꺼운 구조적 구분선, 중요 데이터 하이라이트에 사용.

### 택티컬 텔레메트리 (다크) 선택 시:
- **배경:** `#0A0A0A` 또는 `#121212` (비활성 CRT. 순수 `#000000` 피할 것).
- **전경:** `#EAEAEA` (화이트 인광). 이것이 기본 텍스트 색상.
- **액센트:** `#E61919` 또는 `#FF2A2A` (항공/위험 레드). 동일한 레드, 동일한 규칙.
- **터미널 그린 (`#4AF626`):** 선택 사항. 단일 특정 UI 요소(예: 하나의 상태 표시기 또는 하나의 데이터 판독값)에만 사용 — 일반 텍스트 색상으로는 절대 사용 금지. 명확한 목적이 없으면 완전히 생략.

## 5. 레이아웃 및 공간 엔지니어링

레이아웃은 수학적으로 설계된 것처럼 보여야 한다. 관습적인 웹 패딩을 거부하고 가시적 구획화를 선호한다.

- **청사진 그리드:** CSS Grid 아키텍처의 엄격한 준수. 요소는 부유하지 않고 그리드 트랙과 교차점에 정밀하게 고정된다.
- **가시적 구획화:** 정보의 뚜렷한 영역을 구분하기 위한 단색 테두리 (`1px` 또는 `2px solid`)의 광범위한 활용. 수평선 (`<hr>`)은 작동 단위를 분리하기 위해 전체 컨테이너 너비에 자주 걸친다.
- **이중 밀도:** 레이아웃은 극단적인 데이터 밀도(함께 밀집된 타이트한 모노스페이스 메타데이터)와 매크로 타이포그래피를 프레이밍하는 광대한 계산된 네거티브 스페이스 사이를 오간다.
- **기하학:** `border-radius`의 절대적 거부. 모든 모서리는 기계적 경직성을 강제하기 위해 정확히 90도여야 한다.

## 6. UI 컴포넌트 및 심볼로지

표준 웹 UI 관례는 실용적이고 산업적인 그래픽 요소로 대체된다.

- **구문 장식:** ASCII 문자를 활용한 데이터 포인트 프레이밍.
  - *프레이밍:* `[ DELIVERY SYSTEMS ]`, `< RE-IND >`
  - *방향성:* `>>>`, `///`, `\\\\`
- **산업 마커:** 법적 텍스트가 아닌 구조적 기하학 요소로 기능하는 등록 (`®`), 저작권 (`©`), 상표 (`™`) 기호의 두드러진 통합.
- **기술 에셋:** 그리드 교차점의 크로스헤어 (`+`), 반복 수직선 (바코드), 두꺼운 수평 경고 스트라이프, 활성 기계 프로세스를 시뮬레이션하는 무작위 문자열 데이터 (예: `REV 2.6`, `UNIT / D-01`)의 통합.

## 7. 텍스처 및 후처리 효과

디자인이 순수 디지털로 보이는 것을 방지하기 위해, CSS 및 SVG 필터를 통해 시뮬레이션된 아날로그 열화가 프론트엔드에 엔지니어링된다.

- **하프톤 및 1-Bit 디더링:** 연속 톤 이미지나 대형 세리프 타이포그래피를 도트 매트릭스 패턴으로 변환. 사전 처리 또는 SVG 방사형 도트 패턴과 결합된 CSS `mix-blend-mode: multiply` 오버레이를 통해 달성.
- **CRT 스캔라인:** 터미널 인터페이스의 경우, 수평 전자빔 스윕을 시뮬레이션하기 위해 배경에 `repeating-linear-gradient` 적용 (예: `repeating-linear-gradient(0deg, transparent, transparent 2px, rgba(0,0,0,0.1) 2px, rgba(0,0,0,0.1) 4px)`).
- **기계적 노이즈:** 다크 및 라이트 모드 모두에서 통합된 물리적 그레인을 도입하기 위해 DOM 루트에 적용된 전역 저불투명도 SVG static/noise 필터.

## 8. 웹 엔지니어링 지시

1. **그리드 결정론:** 복잡한 테두리 선언 없이 수학적으로 완벽한 면도날 같은 구분선을 생성하기 위해 대비되는 부모/자식 배경색과 함께 `display: grid; gap: 1px;` 활용.
2. **의미론적 경직성:** 텔레메트리의 기술적 특성을 정확히 반영하기 위해 정밀한 시맨틱 태그 (`<data>`, `<samp>`, `<kbd>`, `<output>`, `<dl>`)를 사용하여 DOM 구축.
3. **타이포그래피 클램핑:** 뷰포트 전반에 걸쳐 구조적 무결성을 유지하면서 대규모 텍스트 스케일이 공격적으로 조정되도록 매크로 타이포그래피에만 CSS `clamp()` 함수 구현.
