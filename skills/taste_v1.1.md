# Taste - Premium Design Quality Skill (v1.1)

> 이 스킬은 프론트엔드/디자인 에이전트에게 주입되어 프리미엄 수준의 UI/UX 품질을 보장합니다.
> v1.1 버전에서는 마이크로 인터랙션과 미려한 전환 효과에 대한 가이드라인이 추가되었습니다.

## 디자인 철학

- Agency-level, interactive, highly polished 디자인을 목표로 한다.
- 일반적인 AI 생성물처럼 보여서는 절대 안 된다.
- 참고 사이트: awwwards.com, land-book.com, lapa.ninja, onepagelove.com, godly.website.

## 디자인 원칙

- 표준적인 Hero + 3 Cards + FAQ 레이아웃을 **금지**한다.
- 프리미엄 레이아웃, 적극적 여백 활용, 다양한 카드/섹션 구조를 사용한다.
- 모던 타이포그래피 (Inter, Outfit, Pretendard 등 시스템 폰트 대신 웹폰트 사용).
- 반응형 디자인과 접근성을 보장한다.
- 컬러 팔레트는 조화롭고, 일반적인 빨강/파랑/초록 원색을 피한다.

## 인터랙션 (v1.1 보강)

- Smooth scroll, micro-interaction 필수.
- Hover 시 scale/shadow 변화.
- Fade-in / Slide-in / Parallax 효과.
- 과도한 애니메이션은 금지 (의미 있는 인터랙션만).
- SVG 애니메이션 및 CSS Keyframe을 활용한 부드러운 요소 전환 보장.

## 미디어

- 고품질 이미지: Unsplash, Pexels, Pixabay 활용.
- 비디오: Mixkit, Coverr 활용.
- 제네릭 placeholder 이미지 사용 금지.

## UI 컴포넌트

- Hero, Features, Screenshots, Testimonials, CTA, Footer를 창의적으로 배치.
- 각 섹션이 고유한 비주얼 아이덴티티를 가져야 한다.

## 코드 품질

- 컴포넌트 기반 구조.
- 시맨틱 HTML, SEO 친화적, 빠른 로딩.
