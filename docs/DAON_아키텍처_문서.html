📌 프로젝트 개요

DAON Agent System v1.2.0은 Hermes Agent 위에 구축된 한국어 1등 멀티에이전트 데스크톱 IDE입니다. Electron + FastAPI + MCP(Model Context Protocol) + 동적 JIT 에이전트 컴파일러로 구성되며, 단일 사용자(1인 대표님) 환경을 위해 설계되었습니다.

🧠 AI 시스템

Dynamic Hermes는 자연어 미션을 받아 자동으로 에이전트 팀을 즉석에서 구성·실행하는 JIT 컴파일러입니다.

1. Planner (미션 분해)

파일: api/dynamic/planner.py · api/dynamic/plan_validator.py

자연어 미션을 받아 DAG(노드 그래프) 형태의 실행 계획으로 변환합니다.

동작 흐름

  1. 사용자 미션 수신
  2. HermesPlanner.plan_mission() 호출
  3. LLM이 JSON 스키마(plan schema)에 맞춰 노드/엣지 생성
  4. validate_plan_schema() + semantic_validate()로 검증
  5. _compute_execution_batches()로 병렬 실행 배치 계산

호출 위치

  • api/dynamic/orchestrator.py:HermesDynamicRunner — 메인 파이프라인 진입점
  • api/routes/chat_routes.py:handle_post_chat_start — 채팅 시작 시
2. AgentCompiler (에이전트 빌드)

파일: api/dynamic/compiler.py

DAG 노드 하나당 실행 가능한 에이전트 객체를 생성합니다. 페르소나/스킬/컨텍스트를 주입합니다.

주입 구성요소

  • Persona — get_integrated_persona() (17 roles 중 선택)
  • Skills — skill_retriever로 매칭된 스킬 자동 첨부
  • Context — _build_node_context() (이전 노드 결과 + 사용자 히스토리)
  • Model — model_selector.py로 노드별 모델 선택
3. ParallelRunner (병렬 실행)

파일: api/dynamic/runner.py

배치별로 노드들을 병렬 실행합니다. 실패 시 _run_node_with_retries()로 자동 재시도합니다.

관련

  • _handle_timeout_node() — 노드 타임아웃 처리
  • _persist_node_result() — 결과 영속화
  • cleanup_harness_artifacts() — 실행 산출물 정리
4. ResultMerger (결과 병합)

파일: api/dynamic/merger.py

여러 노드의 출력을 하나의 최종 응답으로 병합합니다. select_best_output()으로 최우수 결과 선택도 지원합니다.

9. Experience DB

파일: api/dynamic/experience_db.py

10. Style Card System

파일: api/style_card.py · api/dynamic/style_card_extractor.py · api/dynamic/style_card_retriever.py · api/dynamic/style_mixer.py + api/routes/style_card_routes.py

디자인 웹페이지/앱의 DNA를 추출하고, 컴포넌트를 믹싱해 새 디자인 브리프를 생성합니다.

파이프라인

  1. Extract — URL/이미지에서 디자인 DNA 추출
  2. Retrieve — Style Card DB 조회
  3. Mix — 카테고리별 컴포넌트 믹싱 (harmony 검증 포함)
  4. Trend Check — 트렌드 적합성 (deprecated 패턴 감지)
  5. Accessibility — WCAG 2.1 AA 검증

API

  • POST /api/style-cards/extract
  • POST /api/style-cards/save / delete
  • POST /api/style-cards/rebuild-index
